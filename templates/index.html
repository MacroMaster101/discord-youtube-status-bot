Bot Commands Cheat Sheet
These are the official prefix commands available in your Discord guilds. Type your custom prefix (e.g. $) followed by the command inside any server text channel to execute them!

YouTube Feed

$yt stats

Lookup real-time subscriber and view statistics for any YouTube channel.

$yt link

Retrieve the direct URL channel link of the configured primary account.

Information

$stats

Show detailed statistics about the current guild server.

$userinfo

Get account ages, roles, status, and joining dates for any user.

$avatar

Enlarge and view high-quality profile picture for any member.

$servericon

Display the server's primary animated or static guild icon.

$ping

Measure and display the bot's WebSocket heartbeat latency.

$uptime

Check precisely how long the statistics polling engine has been online.
